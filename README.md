# DevTrack - Developer Productivity Dashboard

A modern React dashboard for managing developer tasks with drag-and-drop workflow, dark mode, and local storage persistence.

## Features

- Task manager (add, edit, delete)
- Priority labels (Low, Medium, High)
- Kanban status columns (Todo, In Progress, Completed)
- Drag and drop between columns
- Dark mode toggle
- Local storage persistence
- Smooth animations and hover transitions
- Responsive sidebar + top navigation layout

## Tech

- React (functional components + hooks)
- Vite
- Plain CSS (modern SaaS-inspired design)

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview production build:

```bash
npm run preview
```
